#  Plataforma Educativa 

Plataforma web moderna para cursos online con sistema de becas universitarias.

##  Características

- ✅ Sistema de autenticación (Login/Registro)
- ✅ Registro diferenciado: Becados vs Compradores
- ✅ Catálogo de cursos
- ✅ Programa de becas para universitarios
- ✅ Panel de administración
- ✅ Base de datos MongoDB
- ✅ API REST completa

##  Tecnologías

**Frontend:**
- React 18
- React Router
- Tailwind CSS
- Axios

**Backend:**
- Node.js + Express
- MongoDB + Mongoose
- JWT Authentication
- Bcrypt

##  Instalación

### Backend
```bash
cd backend
npm install
```

Crear archivo `.env`:
```
MONGODB_URI=tu_mongodb_uri
PORT=5000
JWT_SECRET=tu_secreto
```

Ejecutar:
```bash
node server.js
```

### Frontend
```bash
cd frontend
npm install
npm start
```

##  URLs

- Frontend: http://localhost:3000
- Backend: http://localhost:5000

##  Tipos de Usuario

- **Becado**: Estudiante universitario con acceso a becas
- **Comprador**: Usuario regular que adquiere cursos
- **Admin**: Gestiona usuarios, becas y contenido
